Simulador de Compra en un Kiosco

Este simulador permite al usuario realizar una compra simulada en un kiosco. A través de cuadros de diálogo (alert, prompt, confirm) el usuario podrá seleccionar productos disponibles, agregarlos a un carrito virtual, calcular el total de su compra y decidir si desea seguir comprando o finalizar la operación.

El simulador utiliza funcionalidades de JavaScript como variables, constantes, arrays, objetos literales, condicionales (if, else if, else, switch), ciclos de iteración (while, for of), funciones personalizadas y la consola para mostrar mensajes adicionales.
